## Step 1: 选题生成

### 选题方向矩阵（按 10 万+ 概率排序）

| 类型 | 包装角度 | 10万+概率 | 示例 |
|------|---------|----------|------|
| 生肖文化 | 性格+人生哲学 | ★★★★★ | 「属龙的人，骨子里藏着这 3 种性格」 |
| 节气养生 | 节气+养生+民俗 | ★★★★★ | 「谷雨到了，老人常说的 3 个忌讳你知道几个」 |
| 民间智慧 | 老话新解+科学印证 | ★★★★☆ | 「为什么老人说'门前不种三棵树'，原来真有道理」 |
| 家居布局 | 空间哲学+实用建议 | ★★★★☆ | 「客厅摆这个，古人说旺家，现代科学也认同」 |
| 易经哲学 | 经典解读+生活映射 | ★★★☆☆ | 「《易经》里最厉害的一句话，读懂了少走 10 年弯路」 |
| 传统建筑 | 古建筑+空间智慧 | ★★★☆☆ | 「故宫的门为什么都朝南，古人的讲究现代人想不到」 |
| 人物传记 | 历史人物+智慧故事 | ★★★☆☆ | 「袁天罡一生只说了 4 句话，每句都应验了」 |

### 选题规则

1. **节气优先**：当前节气前后 3 天，必须出 1 篇节气内容
2. **生肖轮转**：12 生肖按顺序轮，保证每个生肖每月至少 1 篇
3. **去重检查**：标题相似度 > 60% 跳过
4. 生成 9 个候选 → 按「转发潜力 × 文化深度 × 平台安全」→ Top 3

### 标题公式（专为中老年优化）

```
爆款公式：
  「{生肖}的人，{令人好奇的特质}」          → 社交货币
  「老人常说的'{俗语}'，{揭秘/印证}」       → 怀旧共鸣
  「{节气}到了，{数字}个讲究别忘了」         → 实用刚需
  「{数字}+{好奇心钩子}+{生活相关}」        → 通用公式
  「为什么{现象}？{文化溯源}」              → 求知欲

技巧：
  ├ 数字开头（3 个/5 种/7 大）→ 中老年偏爱明确感
  ├ "老人常说""古人讲究" → 权威感 + 怀旧
  ├ 生肖/属相 → 天然社交货币（"转给属X的朋友看看"）
  └ 节气/时令 → 时效感 + 实用性

⚠️ 标题风控：生成后立即过 Layer 1，不通过则重写
```

## Step 2: 文案生成

### 文风定位（为中老年优化）

```
身份：一个懂传统文化的晚辈，跟长辈聊天的口吻
语气：尊重但不拍马屁，有知识但不卖弄，温暖但不煽情
句式：短句为主，不用长难句，不用英文，不用网络用语
段落：每段不超过 4 行（中老年手机字大，段落短好读）
字号建议：正文 16px+，让排版工具调大

关键原则：
  ├ 用"咱们""您"而不是"你"
  ├ 多用"古人说""老辈人讲" → 权威感
  ├ 每个知识点后面跟一句白话翻译
  └ 结尾一定有"转给{XX}的人看看" → 引导转发
```

### 文案结构模板

```
─── 开头（约 300 字）───

[文化定位声明]
中华文化五千年，很多老祖宗传下来的智慧，
到今天来看，依然有它的道理。

[引入]
今天咱们来聊一个有意思的话题——{主题}。
{一个具体的生活场景或老人常说的一句话}

─── 正文（约 3000-4000 字）───

## {第一个知识点}
{引用原典 + 白话翻译}
{古人怎么做的}
{现代科学/心理学怎么看}

## {第二个知识点}
{同上结构}

## {实用建议}
{读者在生活中可以做什么}
用"古人认为""传统观点"，不用"你应该""你必须"

─── 结尾（约 300 字）───

[总结]
老祖宗的智慧，不是迷信，是千百年来的生活经验总结。
{回到文化传承的高度}

[免责声明]
本文仅为传统文化科普与个人见解分享，不代表任何预测或建议。
中华传统文化博大精深，欢迎理性探讨。

[引导转发 — 核心增长引擎]
觉得有道理的话，转给身边{属X的/关心养生的/喜欢传统文化的}朋友看看。
点个「在看」，让更多人看到老祖宗的智慧。
```

### 文案质量标准

- **字数**：3000-5000 字
- **原典引用**：每篇 ≥ 2 处（《周易》《黄帝内经》《礼记》等）
- **白话翻译**：每处引用后必须跟通俗解释
- **古今对照**：每篇 ≥ 1 处现代科学印证
- **转发钩子**：结尾必须有明确的转发引导语
- **段落长度**：每段 ≤ 4 行
- **禁止**：绝对化预测、付费引导、恐吓话术、网络用语

## Step 3: 风控扫描

**文案生成后，必须过风控扫描。**

```
检查项：
  1. 标题：一级/二级敏感词 → 自动替换
  2. 正文：敏感词扫描 → 替换
  3. 绝对化语句：「必定/一定会/注定」→ 改为转述语
  4. 免责声明：开头文化定位 + 结尾免责 → 缺一不可
  5. 密度检查：三级风险词每 1000 字不超过 3 个
```

```

```

尺寸：4:3（1200×900）→ 中老年手机竖屏友好
风格：与封面统一，国风水墨
内容：与当前章节对应的文化意象

## ⚠️ Pitfalls

1. **标题是最大风险点**：标题触发审核概率远高于正文
2. **不用谐音/生僻字绕审核**：微信加重处罚对抗行为
3. **免责声明不能省**：即使内容很安全，这是最后防线
4. **配图也审核**：图片中的符咒/八卦图同样敏感
5. **不做付费引流**：红线，直接封号
6. **中老年阅读习惯**：段落要短、字要大、不用英文和网络梗
7. **转发引导是增长引擎**：每篇结尾必须有，这是中老年内容的核心增长机制
8. **3 篇/天是测试量**：触发限流就降到 2 篇
